# Future Plans

Tracked feature requests and ideas for Uprooted. Items are listed with who suggested them and current status.

## Feature Requests

### Panel Rearrangement / Layout Customization
**Suggested by:** MrEizy
**Status:** Planned

Allow users to rearrange Root's UI panels:
- Move the server/community list to the left or right side
- Reposition the notifications panel
- Top/bottom placement options for various UI elements

This would require deep DOM manipulation of Root's Avalonia-hosted Chromium layout and likely a dedicated layout engine plugin. The C# hook layer may also need modifications to support native panel repositioning.
